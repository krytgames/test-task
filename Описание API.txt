Использовался mysql server 9.4.0
Использовался laravel 12.33.0

Дополнительно приложил:
1)дамп всех баз данных(на всякий случай)
2)дамп только рабочей базы
3)запросы из своего httpie

1) Для работы необходим токен. Авториция Bearer
Получить токен можно через v1/createToken без параметров

2)v1/createAuthor - создать автора, POST, параметры:
fullName
email
avatar - необязательный

3)v1/createCategory - создать рубрику, POST, параметры:
name
parentID - необязательный

4)v1/createArticle - создать новость, POST, параметры:
authorID
title
anonse
text
categoryIDs[] - массив, необязательный

5)v1/authorsList - список авторов, GET, параметры:
page - необязательный

6)v1/articlesByAuthor - список новостей по автору, GET, параметры:
authorID
page - необязательный

7)v1/articlesByTitle - список новостей по названию, GET, параметры:
page - необязательный
exact - необязательный, Y или N, поиск по точному соответствию
title

Попытался сделать поиск по похожим словам, однако для этого оказалось нужны более продвинутые поисковые движки, а полнотекстовый поиск мне не понравился из-за того, что нельзя ввести часть слова

8)v1/articlesByCategory - список новостей по рубрикам, GET, параметры:
page - необязательный
categoryID
includeSubCategories - искать в под-категориях, Y или N, необязательный

9)v1/articlesByIDs - список новостей по ID, GET, параметры:
page - необязательный
IDs[] - массив
